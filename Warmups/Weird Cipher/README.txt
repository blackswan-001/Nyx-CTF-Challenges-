====================================================
  Weird Cipher
  Category : Cryptography   |   Difficulty : Warmup
====================================================

Every system has a language.
Before you speak to it — you must understand
how it speaks to itself.

Machines do not think in letters.
They think in something older.
Something underneath.

A message has been left for you.
It will not make sense at first.
That is because you are reading it
in the wrong language.

Translate it into something a machine would say.
Then read it again.
